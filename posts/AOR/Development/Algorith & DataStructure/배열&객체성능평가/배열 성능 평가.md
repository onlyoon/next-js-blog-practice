### 배열
----
- 배열은 언제 사용할까?
  - 순서가 필요할 때
  - 입력과 제거시 빠른 접근이 필요할 때
----
- Big O of 배열
  - 삽입: 
    - 배열 끝에서 삽입할 경우 O(1)
    - 배열 앞에서 삽입할 경우 O(N)
    -> 앞에서 접근할 경우 전체 재정리
  - 제거: ??
    - 배열 끝에서 제거할 경우 O(1)
    - 배열 앞에서 제거할 경우 O(N)
    -> 앞에서 접근할 경우 전체 재정리
  - 검색: O(N)
  - 접근: O(1)
