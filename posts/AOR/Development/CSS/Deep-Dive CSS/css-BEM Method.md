### BEM Method
>[!note]
>#### BEM 방법론  
>
>>Block, Element, Modifier를 기준으로 작성하는 네이밍 규칙  

- 📝 예시  
```html  
<div class="list-container__elem--border-red"></div>  
/* 'elem'이라는 이름을 가진 list-container의 클래스의 border는 빨간색이다. */
```  

- 다른 네이밍 방법론  
	- SMACSS, OOCSS
