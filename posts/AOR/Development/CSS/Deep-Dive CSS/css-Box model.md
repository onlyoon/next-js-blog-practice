### Box model
>[!note]
>#### 박스 모델
>
>>HTML의 모든 요소는 CSS에서 박스로 인식된다.
>
>>박스는 3가지의 프로퍼티로 활용이 가능하다. (padding, border, margin)
#### padding
- 안쪽으로 요소를 감싸는 여백
#### border
- 안쪽으로 요소를 감싸는 테두리
#### margin
- 바깥쪽으로 요소를 감싸는 여백

>[!warning]
>>padding 추가시, 요소의 컨텐츠는 해당 콘텐츠에 대한 모든 여백까지의 크기를 포함한다.
>>따라서 padding 추가시, margin에도 같이 포함된다.
>
>>body에는 margin이 있다.
