### Shorthand Properties
>[!note]
>#### 프로퍼티 축약
>
>>여러 다른 프로퍼티의 값을 단일 프로퍼티로 결합된 것을 말한다.

- 📝 예시  
```css  
margin: 5px 10px 5px 10px; /* top, left, bottom, right */  
margin: 5px 10px; /* top & bottom, left & right */  
margin: 10px; /* 전체 적용 */  
```
