### Parameter vs. Argument
>[!note]
>#### 매개변수와 인수
>
>>매개변수: 함수를 정의할 때, 괄호 안에 지정하는 변수
>
>>인수: 함수를 호출할 때, 함수에 전달하는 ==구체적인 값==
#### 매개변수 예시
```js
function sayHi(name) { ... } // <- name 에 해당한다.
```
#### 인수 예시
```js
sayHi('Max'); // <- 'Max'에 해당한다.
```
