### 종류
- LocalStorage
	- 단순한 키-값 쌍을 저장하는 저장소
	- 사용자 설정 || 기본 사용자 데이터
	- 복잡한 데이터를 저장하는 것이 어렵다.
	- 데이터를 수동으로 지워야 한다.
	- 사용자 조작이 가능해 SSOT 가 불가하다.
- SessionStorage
- Cookies
	- 단순한 키-값 쌍을 저장하는 저장소
	- 기본 설정, 세션 ID
	- 복잡한 데이터 저장 X
- IndexedDB
	- 복잡한 데이터 관리가 가능하다.
	- 클라이언트 측 데이터베이스
